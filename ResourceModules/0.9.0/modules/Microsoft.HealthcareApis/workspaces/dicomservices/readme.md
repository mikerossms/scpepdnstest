Tag names and tag values can be provided as needed. A tag can be left without a value.

```json

    "value": {
        "Environment": "Non-Prod",
        "Contact": "test.user@testcompany.com",
        "PurchaseOrder": "1234",
        "CostCenter": "7890",
        "ServiceName": "DeploymentValidation",

}
```

</details>
